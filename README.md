# ✅ To-Do App

A modern and responsive task management application built with **HTML, CSS, and JavaScript**. The app helps users organize their daily tasks with a clean, intuitive interface while demonstrating core frontend development concepts such as DOM manipulation, event handling, and local data persistence.

---

## ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 🔍 Filter tasks (All / Active / Completed)
* 💾 Save tasks using Local Storage
* 📱 Fully Responsive Design
* ⚡ Smooth and user-friendly interactions
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
